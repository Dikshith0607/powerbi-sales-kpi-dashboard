# Amazon Shipment Performance Dashboard (Power BI)

This project presents an interactive **Amazon shipment and sales performance dashboard**
built using public spreadsheet data to analyze order fulfillment, revenue, and delivery status.

The dashboard is designed to simulate a **real-world e-commerce BI reporting use case**
for business and operations stakeholders.

## Business Objective
The objective of this dashboard is to:
- Track overall sales turnover and order volume
- Monitor shipment and delivery performance
- Identify cancellation and pending order trends
- Analyze sales distribution by product size and category
- Support operational and fulfillment decision-making

## Key KPIs
- Total Turnover
- Total Products Sold
- Quantity by Delivery Status
- Revenue by Product Size
- Revenue by Product Category

## Dashboard Highlights
- Interactive slicers for Date, Month, Category, ASIN, Fulfillment type, and Order ID
- Shipment status tracking (Shipped, Delivered, Cancelled, Pending, Returned)
- Category-wise and size-wise revenue analysis
- Drill-down capable table with SKU-level details

## Tools Used
- Power BI
- DAX
- Excel / CSV data source

## Notes
- The dataset used is publicly available and does not contain any confidential or company-specific information.
- This project focuses on **dashboard design, KPI selection, and analytical storytelling** rather than predictive modeling.
